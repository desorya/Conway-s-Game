# Conway-s-Game
Basic implementation of Conway's Game Of Life using glider pattern in Program C.

Source code can be viewed in _**game.h**_ and   _**game.cpp**_.
## Environment

| Program C  |  |
| ------------- | ------------- |
| IDE | Dev-cpp v6.7.5 | 
| Compiler | MinGW GCC 9.2.0 32-bit Profiling |
| Encoder  | UTF-8 |
